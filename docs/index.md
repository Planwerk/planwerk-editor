# Planwerk Editor

A modern documentation viewer and editor built with Next.js, featuring Claude Code session support and real-time markdown editing capabilities.

## Overview

This project provides a powerful documentation platform that combines a rich markdown editor with an integrated terminal, making it easy to maintain and update your documentation while having direct access to development tools.

## Features

- **Real-time Markdown Editing**: Edit your documentation with instant preview
- **Integrated Terminal**: Execute commands directly from the documentation interface
- **Claude Code Integration**: Seamless support for Claude Code sessions
- **Modern Stack**: Built with Next.js 15, TypeScript, and Tailwind CSS
- **File Watching**: Automatic updates when documentation files change

## Getting Started

Navigate through the documentation using the sidebar, or use the built-in editor to create and modify content. The integrated terminal provides quick access to development commands without leaving the documentation interface.
